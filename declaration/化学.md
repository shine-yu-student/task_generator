# 化学题型命制方法

## 科目概述

新高考化学试卷结构（100 分）：
- 选择题 7 道（每题 6 分，共 42 分，均为单选）
- 非选择题 3 道（共 58 分）

易:中:难 ≈ 5:3:2

---

## 一、选择题

### 常考题型及命制方法

### 1.1 化学基本概念与常考判断型

**命题方式**：给出 4 个化学说法，选出正确/错误的。

**常见考点**：
- 物质的组成与分类
- 氧化还原反应判断
- 离子共存/离子方程式正误
- 阿伏加德罗常数
- 化学键类型判断

**命制要点**：
- 干扰项来源于常见误解
- 4 个选项覆盖不同知识领域或不同角度
- 确保选项的独立性和互斥性

### 1.2 元素推断型

**命题方式**：给出短周期元素信息（原子序数、位置关系、性质），推断元素并回答问题。

**命制方法**：
1. 给出 2~4 种元素的结构/位置/性质线索
2. 要求判断元素符号
3. 考查化合物性质、反应类型等

### 1.3 有机化学选择题

**命题方式**：给出有机物结构，判断同分异构体数目、反应类型、化学性质。

**命制方法**：
1. 给出有机物结构简式
2. 判断能发生的反应类型
3. 同分异构体计数
4. 有机物命名或官能团判断

### 1.4 电化学型

**命题方式**：给出原电池/电解池装置，分析电极反应、离子移动、计算。

**命制方法**：
1. 描述电池装置（正负极材料、电解质溶液）
2. 考查电极反应式书写
3. 考查离子迁移方向
4. 考查电子转移与计算

### 1.5 化学平衡型

**命题方式**：给出反应体系，判断平衡移动方向、转化率变化、浓度变化。

**命制方法**：
1. 描述可逆反应条件
2. 改变条件（温度/压强/浓度）
3. 用勒夏特列原理判断平衡移动
4. 图象判断（浓度-时间、转化率-温度图）

### 1.6 水溶液型

**命题方式**：考查离子浓度大小比较、pH 计算、盐类水解。

**命制方法**：
1. 给出溶液体系
2. 判断离子浓度大小排序
3. 溶液酸碱性及 pH 计算
4. 水解/电离平衡分析

### 1.7 化学实验基础型

**命题方式**：考查实验操作、仪器使用、安全常识。

**命制方法**：
1. 给出实验操作描述
2. 判断操作正误
3. 考查装置气密性检查、除杂方法
4. 实验安全与事故处理

## 二、非选择题

### 2.1 工艺流程题（约 14 分）

**题型特征**：给出从矿石/原料到产品的工业生产流程图，考查反应原理和操作分析。

**命制方法**：
1. **选材**：选择真实的工业生产背景（如从钛铁矿提取钛、从废催化剂回收金属）
2. **设计流程**：
   - 原料预处理（粉碎、焙烧）
   - 溶解/浸出（酸溶/碱溶）
   - 除杂（调节 pH、沉淀）
   - 核心反应（氧化还原、配合）
   - 产品分离（蒸发结晶、过滤）
3. **设问方式**：
   - 焙烧/溶解目的
   - 调节 pH 的范围及原因
   - 反应方程式书写
   - 沉淀成分判断
   - 产品纯度计算
   - 循环利用物质判断

**命制要点**：
- 流程不宜过于复杂，3~5 步核心操作
- 设问由浅入深：操作目的 → 方程式 → 计算分析
- 关键步骤需在流程图中标注清楚

### 2.2 化学实验综合题（约 14 分）

**题型特征**：给出实验装置和目的，考查实验设计、操作分析、数据处理。

**命制方法**：
1. **实验类型**：
   - 气体制备与性质验证
   - 物质含量测定（滴定）
   - 反应条件探究
   - 物质检验与鉴别
2. **设问方式**：
   - 装置连接顺序/补充
   - 试剂选择及作用
   - 实验操作及注意事项
   - 方程式书写
   - 误差分析
   - 数据处理与计算

**命制要点**：
- 实验目的明确
- 装置描述清晰（可用文字描述代替图片）
- 滴定类需给出数据表格

### 2.3 反应原理综合题（约 14 分）

**题型特征**：综合考查盖斯定律、化学平衡、电化学等原理。

**命制方法**：
1. **热化学**：利用盖斯定律计算反应焓变
2. **化学平衡**：
   - 计算平衡常数 K
   - 判断反应自发性
   - 平衡转化率计算
3. **电化学**：
   - 设计电池
   - 电极反应式
   - 计算电量/质量

**命制要点**：
- 分小问，每问考查一个原理
- 计算过程给出必要数据
- 图象题需描述清楚坐标轴含义

### 2.4 有机化学综合题（约 14 分，部分省份选考）

**题型特征**：给出有机合成路线，推断结构、书写方程式、设计合成路线。

**命制方法**：
1. **合成路线设计**：从原料到目标产物，经过 3~5 步反应
2. **设问方式**：
   - 推断中间产物结构简式
   - 判断反应类型（取代/加成/消去/氧化/还原/酯化/水解）
   - 书写化学方程式
   - 同分异构体书写
   - 设计合成路线
3. **命制要点**：
- 合成路线逻辑清晰，每步反应均为经典反应类型
- 结构推断线索充分
- 同分异构体要求明确（限制条件）

## 年级难度梯度

| 年级 | 难度 | 覆盖范围 |
|------|------|----------|
| 高一 | 基础 | 物质分类、氧化还原、离子反应、钠铝铁硅、氯硫氮 |
| 高一 | 提高 | 化学计算综合、元素化合物推断 |
| 高二 | 基础 | 化学平衡、电离水解、电化学基础、有机化学基础 |
| 高二 | 提高 | 平衡综合计算、有机推断、实验设计 |
| 高三 | 高考 | 全模块综合、工艺流程、实验综合 |

## JSON 生成提示

1. 指明科目 (`chemistry`)、年级、难度
2. 选择题指明考查类型（概念判断/元素推断/有机/电化学/平衡/水溶液/实验基础）
3. 非选择题指明类型（工艺流程/实验综合/反应原理/有机综合）
4. 化学方程式用 LaTeX 格式，如 $\ce{2Na + 2H2O -> 2NaOH + H2 ^}$ 或纯文本 `2Na + 2H₂O → 2NaOH + H₂↑`
5. 工艺流程题需用文字清晰描述流程步骤
6. 有机物结构用系统命名或结构简式描述
7. 计算题需给出完整数据和解题步骤
8. 填空题的答案需列出所有可接受的等价写法
