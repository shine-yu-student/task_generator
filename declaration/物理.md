# 物理题型命制方法

## 科目概述

新高考物理试卷结构（100 分）：
- 单选题 4~6 道（每题 4~6 分）
- 多选题 4~6 道（每题 6 分，部分选对得 3 分）
- 实验题 2 道（共 14~16 分）
- 计算题 2 道（共 36~40 分）

易:中:难 ≈ 5:3:2

---

## 一、单选题

### 命制方法

1. **概念理解型**：考查物理概念和规律的理解（如"关于牛顿第一定律，下列说法正确的是"）
2. **图象分析型**：给出物理量关系图象（v-t、F-a、U-I 等），分析物理过程
3. **定性判断型**：不涉及复杂计算，通过推理判断结果
4. **估算型**：对物理量进行数量级估算

### 命制要点

- 每题考查 1~2 个知识点
- 选项避免出现"以上都对/都不对"
- 干扰项来源于常见错误概念

### 覆盖模块

力学：运动学、牛顿定律、曲线运动、万有引力、机械能
电磁学：静电场、恒定电流、磁场、电磁感应
热学、光学、原子物理（选修部分）

---

## 二、多选题

### 命制方法

1. **多结论判断型**：对同一物理情境得出多个结论，选出正确的
2. **过程分析型**：分析物理过程中各量变化，选出正确的判断
3. **多条件分析型**：在不同条件下分析同一物理规律

### 命制要点

- 至少 2 个正确选项
- 部分选对得 3 分，有错选 0 分
- 正确选项间应有逻辑关联，非随意组合

---

## 三、实验题

### 命制方法

### 3.1 力学实验

常见实验：
- **验证牛顿第二定律**：控制变量法、a-F/a-m 图象
- **验证机械能守恒定律**：打点计时器、纸带分析
- **探究平抛运动**：轨迹分析
- **验证动量守恒**：碰撞实验

命制方式：
1. 给出实验装置图描述，考查实验原理
2. 填写实验步骤排序/补充
3. 读数填空（游标卡尺、螺旋测微器）
4. 误差分析
5. 数据处理（作图、计算）

### 3.2 电学实验

常见实验：
- **测定金属电阻率**：伏安法、电路选择
- **描绘小灯泡伏安特性曲线**：分压/限流电路
- **测定电源电动势和内阻**：U-I 图象
- **半偏法测电阻**：电表改装

命制方式：
1. 电路设计/电路补全
2. 器材选择（量程、分压限流）
3. 数据处理与误差分析
4. 实验方案评价与改进

### 命制要点

- 实验题需给出充分的实验背景描述
- 分步设问，由操作到分析到评价
- 填空处答案应明确唯一
- 读数类需明确精度要求

---

## 四、计算题

### 4.1 力学综合题（第一道计算题，较基础）

### 命制方法

1. **运动学综合**：匀变速运动 + 平抛/圆周
2. **牛顿定律综合**：连接体、传送带、板块模型
3. **能量综合**：动能定理 + 机械能守恒

### 命制要点

- 分 2~3 小问，由易到难
- 第(1)问通常为基本公式应用
- 后续问综合多个物理过程

### 4.2 电磁学综合题（压轴计算题）

### 命制方法

1. **带电粒子在电磁场中运动**：电场加速 + 磁场偏转
   - 画轨迹 → 找圆心 → 求半径/周期
2. **电磁感应综合**：导体棒切割/线框进出磁场
   - 电动势 → 电流 → 安培力 → 运动分析
3. **力电综合**：力学与电磁学结合

### 命制要点

- 需给出清晰的物理情境描述（含关键数值）
- 分步设问，每问有独立得分点
- 最后一问通常为最值/范围/临界问题
- 需标注每问分值

## 年级难度梯度

| 年级 | 难度 | 覆盖范围 |
|------|------|----------|
| 高一 | 基础 | 运动学、牛顿定律、曲线运动基础 |
| 高一 | 提高 | 万有引力、机械能守恒综合 |
| 高二 | 基础 | 静电场、恒定电流、磁场基础 |
| 高二 | 提高 | 电磁感应、交变电流、热学光学 |
| 高三 | 高考 | 全模块综合、实验设计、压轴计算 |

## JSON 生成提示

1. 指明科目 (`physics`)、年级、难度
2. 指明题型（单选/多选/实验/计算）和知识模块
3. 物理公式用 LaTeX 格式
4. 实验题需描述实验装置和原理
5. 计算题需给出清晰物理情境和数值
6. 计算题分小问标注分值
7. 多选题确保至少 2 个正确选项
8. 需给出标准答案和详细解析过程
