# 生物题型命制方法

## 科目概述

新高考生物试卷结构（100 分）：
- 选择题 15 道（每题 2~3 分，共 36~45 分，均为单选）
- 非选择题 4~5 道（共 55~64 分，以填空+简答为主）

易:中:难 ≈ 5:3:2

---

## 一、选择题

### 常考题型及命制方法

### 1.1 概念理解型

**命题方式**：考查生物学核心概念的理解和辨析。

**常见考点**：
- 细胞结构与功能
- 细胞膜的功能特性（选择透过性 vs 流动性）
- 有丝分裂 vs 减数分裂
- DNA 复制、转录、翻译
- 内环境稳态
- 生态系统的结构与功能

**命制要点**：
- 干扰项来自常见概念混淆
- 考查"理解"而非"记忆"——选项不能直接从教材照搬
- 注意概念的严谨性，避免歧义

### 1.2 实验分析型

**命题方式**：给出实验方案或实验结果，分析实验原理或结论。

**命制方法**：
1. 描述实验目的和步骤
2. 给出实验结果（数据/现象）
3. 判断实验结论/分析实验设计

**常见考点**：
- 渗透作用实验
- 酶活性实验
- 光合/呼吸作用实验
- DNA 半保留复制实验
- 探究性实验与验证性实验的区别

### 1.3 图表分析型

**命题方式**：给出曲线图/柱状图/表格，分析生物学规律。

**命制方法**：
1. 给出坐标图或数据表
2. 考查数据趋势分析
3. 判断变量关系
4. 分析原因

**常见考点**：
- 种群数量变化曲线
- 光合作用强度曲线
- 酶活性曲线
- 细胞分裂各期染色体/DNA 变化
- 生态系统能量流动数据

### 1.4 情境应用型

**命题方式**：以生活/科研/生产情境为背景，考查知识迁移。

**命制方法**：
1. 给出真实的科研或生活情境
2. 提取生物学原理
3. 应用原理解释现象

## 二、非选择题（大题）

### 2.1 细胞代谢综合题

**题型特征**：围绕光合作用和呼吸作用，考查过程、影响因素、实验分析。

**命制方法**：
1. 给出情境材料（如大棚种植、作物产量分析）
2. 分设小问：
   - 光合/呼吸过程填空
   - 影响因素分析（光照强度、CO₂ 浓度、温度）
   - 实验设计与分析
   - 曲线图分析
3. 设问方式：填空题为主，偶有简答

**命制要点**：
- 过程描述需准确使用专业术语
- 填空答案简洁明确
- 实验设计需遵循对照原则

### 2.2 遗传变异综合题

**题型特征**：考查遗传规律的应用，常为试卷难度最高的题目之一。

**命制方法**：
1. **基础型**：一对/两对相对性状的遗传分析
   - 给出亲本表现型和子代表现型比例
   - 判断显隐性关系和基因型
   - 计算后代概率

2. **综合型**：
   - 伴性遗传 + 常染色体遗传
   - 致死基因
   - 多基因/上位效应
   - 基因频率计算

3. **设问方式**：
   - 判断显隐性及基因位置
   - 写出亲本基因型
   - 计算后代某表现型概率
   - 遗传系谱图分析
   - 实验设计方案（验证基因位置）

**命制要点**：
- 遗传背景描述需完整清晰
- 概率计算给出最终分数形式
- 实验方案需写出杂交组合和预期结果

### 2.3 生命活动调节综合题

**题型特征**：考查神经调节、体液调节、免疫调节。

**命制方法**：
1. 给出生理调节情境
2. 考查：
   - 反射弧组成
   - 神经递质传递
   - 激素调节机制（分级调节、反馈调节）
   - 血糖调节
   - 免疫过程（体液免疫/细胞免疫）
3. 设问方式：填空 + 简答

### 2.4 生态学综合题

**题型特征**：考查种群、群落、生态系统的结构与功能。

**命制方法**：
1. 给出生态系统情境
2. 考查：
   - 种群特征与数量变化
   - 群落演替
   - 食物网/食物链分析
   - 能量流动计算
   - 物质循环
   - 生态平衡与稳定性
3. 设问方式：填空 + 简答 + 计算

### 2.5 实验设计与评价题

**题型特征**：独立实验设计或对给定实验的评价改进。

**命制方法**：
1. 给出实验课题
2. 要求：
   - 提出假设
   - 设计实验步骤
   - 预期结果
   - 对已有实验方案进行评价和改进

**命制要点**：
- 遵循单一变量原则、对照原则、重复性原则
- 答案需包含完整的实验步骤
- 给出评分标准时按要点赋分

## 年级难度梯度

| 年级 | 难度 | 覆盖范围 |
|------|------|----------|
| 高一 | 基础 | 细胞结构与功能、细胞代谢、细胞分裂 |
| 高一 | 提高 | 光合呼吸综合、有丝分裂实验分析 |
| 高二 | 基础 | 遗传规律、变异、进化、稳态调节基础 |
| 高二 | 提高 | 遗传综合题、调节综合、生态分析 |
| 高三 | 高考 | 全模块综合、实验设计、遗传推断 |

## JSON 生成提示

1. 指明科目 (`biology`)、年级、难度
2. 选择题指明考查类型（概念/实验/图表/情境）
3. 非选择题指明模块（细胞代谢/遗传/调节/生态/实验设计）
4. 生物学术语必须准确（如"内环境稳态"而非"体内平衡"）
5. 遗传题需给出完整的遗传背景描述
6. 填空题答案需列出可接受的等价表述
7. 实验设计题需提供完整的参考答案（含实验步骤）
8. 生态学计算需给出计算过程
