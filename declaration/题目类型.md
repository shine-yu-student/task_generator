# 题目类型定义

本文档定义生成器支持的所有基本题目类型、其 JSON 格式及前端解析/渲染方法。

---

## 1. 单选题 (single_choice)

**描述**：给出题干和 4 个选项，仅有 1 个正确答案。

**JSON 格式**：
```json
{
  "type": "single_choice",
  "text": "题干文本",
  "options": ["选项A", "选项B", "选项C", "选项D"],
  "answer": 0
}
```

**字段说明**：
- `options`: 字符串数组，长度为 4
- `answer`: 正确选项的索引（0-3）

**解析方法**：
- 渲染为可点击的 ABCD 按钮组，单选互斥
- 提交时将用户选中索引与 `answer` 比较
- 未选择视为未答

---

## 2. 多选题 (multiple_choice)

**描述**：给出题干和 4 个选项，有 1~4 个正确答案（至少 2 个正确）。

**JSON 格式**：
```json
{
  "type": "multiple_choice",
  "text": "题干文本",
  "options": ["选项A", "选项B", "选项C", "选项D"],
  "answers": [0, 2]
}
```

**字段说明**：
- `options`: 字符串数组，长度为 4
- `answers`: 正确选项的索引数组（长度 2~4）

**解析方法**：
- 渲染为可多选的 ABCD 按钮组，选中可取消
- 提交时将用户选中的索引集合与 `answers` 集合比较
- 全部选对且无多余选择 = 正确；否则 = 错误
- 可选：部分选对得一半分（高考多选评分规则）

---

## 3. 填空题 (fill_blank)

**描述**：题干中含有一个或多个空白处（用 `___` 标记），需填写答案。

**JSON 格式**：
```json
{
  "type": "fill_blank",
  "text": "已知函数 f(x) = x² + ___，则 f(2) = ___",
  "blanks": [
    {
      "index": 0,
      "acceptable_answers": ["3", "三"],
      "explanation": "由题意可得 a=3"
    },
    {
      "index": 1,
      "acceptable_answers": ["7", "七"],
      "explanation": "f(2)=4+3=7"
    }
  ]
}
```

**字段说明**：
- `text`: 题干文本，空白用 `___` 标记，按出现顺序编号
- `blanks`: 数组，每项对应一个空白
- `acceptable_answers`: 可接受的答案列表（支持多种等价写法）
- `explanation`: 该空的解析说明

**解析方法**：
- 将 `___` 替换为 `<input>` 框渲染
- 提交时逐空比对用户输入与 `acceptable_answers`（忽略首尾空格，大小写视科目配置决定是否敏感）
- 可选：支持正则匹配

---

## 4. 判断题 (true_false)

**描述**：给出一个陈述，判断其正误。

**JSON 格式**：
```json
{
  "type": "true_false",
  "text": "DNA复制的方向是5'→3'",
  "answer": true,
  "explanation": "DNA复制沿5'→3'方向进行，这是由DNA聚合酶的特性决定的"
}
```

**字段说明**：
- `answer`: 布尔值，true = 正确，false = 错误
- `explanation`: 解析说明

**解析方法**：
- 渲染为 "正确 ✓ / 错误 ✗" 两个按钮
- 提交时比对布尔值

---

## 5. 简答题 / 主观题 (short_answer)

**描述**：开放性题目，无固定答案，需手动评分或参考答案自评。

**JSON 格式**：
```json
{
  "type": "short_answer",
  "text": "请简述光合作用的整个过程",
  "reference_answer": "光合作用分为光反应和暗反应两个阶段……",
  "keywords": ["光反应", "暗反应", "ATP", "NADPH", "Calvin循环"],
  "scoring_criteria": "答出光反应和暗反应各得2分，提及ATP和NADPH各得1分，共6分"
}
```

**字段说明**：
- `reference_answer`: 参考答案
- `keywords`: 关键词列表，可用于辅助自动评分
- `scoring_criteria`: 评分标准描述

**解析方法**：
- 渲染为 `<textarea>` 输入区
- 提交后展示参考答案和关键词，用户自评
- 可选：关键词命中数辅助评分提示

---

## 6. 计算题 (calculation)

**描述**：含计算过程的题目，常见于数学、物理。需填写最终答案，可选要求写出过程。

**JSON 格式**：
```json
{
  "type": "calculation",
  "text": "求不定积分 ∫(2x + 1)dx",
  "final_answer": "x² + x + C",
  "acceptable_answers": ["x^2+x+C", "x²+x+C", "x² + x + C"],
  "steps": [
    {"content": "将积分拆分为 ∫2x dx + ∫1 dx", "score": 2},
    {"content": "分别积分得 x² + x", "score": 2},
    {"content": "加上常数C", "score": 1}
  ],
  "total_score": 5
}
```

**字段说明**：
- `final_answer`: 标准最终答案
- `acceptable_answers`: 可接受的最终答案列表（等价写法）
- `steps`: 解题步骤及每步分值
- `total_score`: 总分

**解析方法**：
- 渲染题干 + 答案输入框 + 过程输入区
- 提交后展示完整解题步骤和最终答案
- 最终答案比对参照 `acceptable_answers`

---

## 7. 排序题 (ordering)

**描述**：将若干条目按正确顺序排列。

**JSON 格式**：
```json
{
  "type": "ordering",
  "text": "请将以下实验步骤按正确操作顺序排列",
  "items": ["称量药品", "溶解", "过滤", "蒸发结晶"],
  "correct_order": [0, 1, 2, 3]
}
```

**字段说明**：
- `items`: 待排序的条目列表（打乱顺序呈现）
- `correct_order`: 正确顺序的索引数组

**解析方法**：
- 渲染为可拖拽的条目列表
- 提交时比对用户排列顺序与 `correct_order`

---

## 8. 匹配题 (matching)

**描述**：将左右两组条目进行配对。

**JSON 格式**：
```json
{
  "type": "matching",
  "text": "将化学元素与其符号配对",
  "left": ["氧", "碳", "氮", "氢"],
  "right": ["N", "C", "O", "H"],
  "correct_pairs": {"0": 2, "1": 1, "2": 0, "3": 3}
}
```

**字段说明**：
- `left` / `right`: 左右两组条目
- `correct_pairs`: 键值对，key 为 left 索引（字符串），value 为 right 索引

**解析方法**：
- 左侧每条渲染一个下拉框，选项为 right 内容
- 提交时比对每个配对与 `correct_pairs`

---

## 9. 完形填空 (cloze)

**描述**：一段文本中挖去若干词，每空给出选项。常见于英语完形填空。

**JSON 格式**：
```json
{
  "type": "cloze",
  "text": "She ___1___ to school every day. She ___2___ hard.",
  "blanks": [
    {
      "index": 1,
      "options": ["go", "goes", "going", "went"],
      "answer": 1
    },
    {
      "index": 2,
      "options": ["study", "studies", "studying", "studied"],
      "answer": 1
    }
  ]
}
```

**字段说明**：
- `text`: 含编号空白标记 `___N___` 的文本
- `blanks`: 每空对应选项及正确答案索引

**解析方法**：
- 将 `___N___` 替换为下拉选择框，选项为对应 blank 的 options
- 提交时逐空比对 answer

---

## 10. 七选五 (seven_choose_five)

**描述**：一段文章挖去 5 个句子，从 7 个备选项中选择 5 个填入。常见于英语阅读七选五。

**JSON 格式**：
```json
{
  "type": "seven_choose_five",
  "text": "文章内容，其中用 ___1___ ___2___ ... ___5___ 标记空位",
  "options": ["A. 备选项1", "B. 备选项2", "C. 备选项3", "D. 备选项4", "E. 备选项5", "F. 备选项6", "G. 备选项7"],
  "answers": {"1": 0, "2": 3, "3": 5, "4": 1, "5": 4}
}
```

**字段说明**：
- `options`: 7 个备选项
- `answers`: 键值对，key 为空位编号（字符串），value 为 options 索引

**解析方法**：
- 每个空位渲染下拉框，选项为 A~G
- 提交时逐空比对

---

## 11. 读后续写 / 写作题 (writing)

**描述**：给出情景材料，要求续写或写作。纯主观题。

**JSON 格式**：
```json
{
  "type": "writing",
  "text": "阅读下面短文，根据其内容和所给段落开头语续写两段……",
  "prompt_paragraphs": ["Para 1: ...", "Para 2: ..."],
  "word_count": {"min": 100, "max": 150},
  "reference_answer": "参考范文……",
  "scoring_criteria": "内容要点5分、语言表达5分、篇章结构5分……"
}
```

**字段说明**：
- `prompt_paragraphs`: 续写段落开头语
- `word_count`: 字数要求
- `reference_answer`: 参考范文
- `scoring_criteria`: 评分标准

**解析方法**：
- 渲染题干 + `<textarea>` 写作区 + 字数统计
- 提交后展示参考范文和评分标准，自评

---

## 12. 语法填空 (grammar_cloze)

**描述**：一段文本，部分空格不给选项，需根据语法填写单词或词形变化。

**JSON 格式**：
```json
{
  "type": "grammar_cloze",
  "text": "The boy ___1___ (run) to school when he ___2___ (see) his teacher.",
  "blanks": [
    {
      "index": 1,
      "answer": "was running",
      "acceptable_answers": ["was running", "was runing"],
      "hint": "run"
    },
    {
      "index": 2,
      "answer": "saw",
      "acceptable_answers": ["saw"],
      "hint": "see"
    }
  ]
}
```

**字段说明**：
- `text`: 含编号空白标记的文本，括号内为提示词
- `hint`: 原词提示（可选）
- `acceptable_answers`: 可接受的答案列表

**解析方法**：
- 将 `___N___` 替换为输入框，括号提示词显示在旁
- 提交时逐空比对，忽略首尾空格，大小写不敏感

---

## 通用包装格式

一份完整的试卷/题目组由以下 JSON 结构包装：

```json
{
  "subject": "math",
  "grade": "高三",
  "difficulty": "高考",
  "topic": "三角函数",
  "questions": [
    { "type": "single_choice", ... },
    { "type": "fill_blank", ... },
    { "type": "calculation", ... }
  ]
}
```

**字段说明**：
- `subject`: 科目标识，取值 `math` / `physics` / `chemistry` / `biology` / `english`
- `grade`: 年级，取值 `高一` / `高二` / `高三`
- `difficulty`: 难度，取值 `基础` / `提高` / `高考`
- `topic`: 知识点/题材
- `questions`: 题目数组，每题含 `type` 字段用于分发渲染
