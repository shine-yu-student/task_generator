# 数学题型命制方法

## 科目概述

高考数学（新高考）试卷结构：
- 单选题 8 道（每题 5 分，共 40 分）
- 多选题 3 道（每题 6 分，共 18 分，部分选对得 3 分）
- 填空题 3 道（每题 5 分，共 15 分）
- 解答题 5 道（共 77 分）

易:中:难 ≈ 5:3:2

---

## 一、单选题

### 命制方法

1. **概念辨析型**：针对易混概念设置选项，如"以下关于函数奇偶性的说法正确的是"
2. **计算求解型**：给出具体条件，要求计算结果，如求值、求范围
3. **图象判断型**：给出函数/几何图形条件，判断图象特征
4. **命题判断型**：给出 4 个命题，选出正确/错误的

### 命制要点

- 干扰项应具有合理性，来源于常见错误
- 正确选项位置均匀分布
- 避免出现"以上都不对"类选项
- 题干简洁明确，避免歧义

### 覆盖知识模块

集合与逻辑、函数与导数、三角函数、数列、立体几何、解析几何、概率统计、向量

---

## 二、多选题

### 命制方法

1. **性质判断型**：给出 4 个关于某概念/定理的命题，选出所有正确的（如"关于等差数列的性质，下列正确的是"）
2. **条件判断型**：给出一个命题和 4 个充分/必要条件
3. **多结论型**：给出一个情境，导出多个结论，选出正确的

### 命制要点

- 至少 2 个正确选项
- 错误选项应有较强迷惑性
- 考查多角度理解，非简单堆砌
- 部分选对得分机制：全对 6 分，部分对 3 分，有错 0 分

---

## 三、填空题

### 命制方法

1. **直接计算型**：计算后填写结果（如求函数值、概率值）
2. **推理填写型**：根据条件推理得出结论
3. **开放型**：给出条件，填写满足条件的值（多个答案均可）

### 命制要点

- 答案唯一或可枚举（开放题需列出所有可接受答案）
- 注意填空题没有过程分，答案必须精确
- 数值答案需注明是否有单位要求

---

## 四、解答题（计算/证明）

### 命制方法

1. **三角函数与解三角形**：给定三角形条件或三角恒等式，求角/边/最值
   - 第(1)问通常为基础计算
   - 第(2)问常涉及最值或范围

2. **数列**：求通项公式、前 n 项和，证明数列性质
   - 常用方法：错位相减法、裂项相消法、分组求和法

3. **立体几何**：证明线面关系 + 求角/距离
   - 第(1)问证明平行/垂直
   - 第(2)问建系求二面角/线面角

4. **概率统计**：分布列、期望、回归分析、独立性检验
   - 情境化命题（实际应用场景）

5. **解析几何**：椭圆/抛物线/双曲线 + 直线
   - 第(1)问求方程
   - 第(2)问求最值/范围/定点/定值

6. **导数综合**：函数零点、不等式证明、极值点偏移
   - 压轴题，综合性最强
   - 常见方法：分类讨论、分离参数、构造函数

### 命制要点

- 分层设问，由易到难
- 第(1)问通常为基础得分点
- 后续问题逐步提升综合度
- 每问须有明确评分标准
- 注意给出合理的分值分配（如 12 分题：4+4+4 或 4+8）

## 年级难度梯度

| 年级 | 难度 | 覆盖范围 |
|------|------|----------|
| 高一 | 基础 | 集合、函数、三角函数、数列基础、立体几何初步 |
| 高一 | 提高 | 函数综合、三角变换、数列求和、空间几何 |
| 高二 | 基础 | 导数初步、概率统计、解析几何、向量 |
| 高二 | 提高 | 导数综合、圆锥曲线、条件概率、空间向量 |
| 高三 | 高考 | 全模块综合、压轴难度 |

## JSON 生成提示

生成数学题目时应遵循以下 prompt 规则：

1. 指明科目 (`math`)、年级、难度
2. 指明题型和知识模块
3. 数学公式用 LaTeX 行内格式 `$...$` 或行间格式 `$$...$$`
4. 解答题必须分小问，标注每问分值
5. 填空题需列出所有可接受的等价答案
6. 多选题需确保至少 2 个正确选项

## GCLC 几何图形嵌入

对于涉及几何图形的题目（如三角形、圆、直线、多边形、坐标系中的函数曲线等），必须同时在 JSON 中给出对应的 GCLC 代码，填入 `"gclc"` 字段（字符串）。该代码用于在题面中渲染几何图形。

### 何时提供 GCLC 代码
- 题目涉及三角形（如解三角形、求角、求边、证明几何关系）
- 题目涉及圆（如圆的方程、直线与圆的关系、内切圆/外接圆）
- 题目涉及立体几何图形（在平面上的投影/示意图）
- 题目涉及解析几何（椭圆、抛物线、双曲线）
- 题目涉及坐标系中的函数图像（可用坐标系统命令绘制）
- 纯计算、纯代数、概率统计题不需要 GCLC 代码

### GCLC 语言简要参考

GCLC 是对几何构造的形式化描述语言。坐标原点 (0,0) 在左下角，默认画布 140mm×100mm。以下是常用命令：

**基本定义与构造：**
```
point A 50 45          % 定义固定点 A，坐标为 (50,45)（单位：mm）
line l A B             % 通过 A、B 两点定义直线 l
circle k O A           % 以 O 为圆心、过 A 定义圆 k
intersec P l1 l2       % 求两直线 l1 和 l2 的交点 P
midpoint M A B         % 求线段 AB 的中点 M
med s A B              % 作线段 AB 的垂直平分线
bis s A B C            % 作角 ABC 的角平分线
perp p A l             % 过点 A 作直线 l 的垂线
parallel p A l         % 过点 A 作直线 l 的平行线
translate A' X Y A     % 将 A 按向量 XY 平移得到 A'
rotate A' O 90 A       % 将 A 绕 O 旋转 90° 得到 A'
towards P A B 0.5      % 在线段 AB 上取点 P 使 AP = 0.5·AB
foot F A l             % 求点 A 到直线 l 的垂足
```

**绘图与标记：**
```
drawsegment A B        % 绘制线段 AB
drawline l             % 绘制直线 l（或 drawline A B）
drawcircle k           % 绘制圆 k（或 drawcircle O A）
drawdashsegment A B    % 绘制虚线线段
drawdashcircle O A     % 绘制虚线圆
drawpoint A            % 绘制点 A
cmark_lt A             % 标记点 A（小圆圈）并在左上方标注 "A"
cmark_lb B             % 标记点 B（左下方标注）
cmark_rt C             % 标记点 C（右上方标注）
cmark_rb D             % 标记点 D（右下方标注）
cmark_t E              % 标记点 E（上方标注）
cmark_b F              % 标记点 F（下方标注）
mark_lt X              % 仅标注（不画圆圈）
```

**填充与样式：**
```
filltriangle A B C     % 填充三角形
fillcircle O A         % 填充圆
color 255 0 0          % 设置颜色 (R,G,B，0-255)
linewidth 0.5          % 设置线宽（mm）
dash 2                 % 设置虚线每段长度（mm）
```

**坐标系（用于函数图像）：**
```
ang_picture 0 0 140 100   % 设置画布范围
ang_origin 20 50          % 设置坐标原点位置
ang_drawsystem            % 绘制坐标轴与刻度
ang_point P 2 3           % 在坐标系中以 (2,3) 定义点 P
```

### 简单示例

题目："已知三角形 ABC 中，AB=5, AC=6, BC=7，求角 A 的大小。"对应 GCLC：

```
point A 50 65
point B 30 30
point C 90 30

cmark_lt A
cmark_lb B
cmark_rb C

drawsegment A B
drawsegment B C
drawsegment C A
```

### 注意事项
- 固定点（通过坐标定义的点）应尽量少（通常 2-4 个），其余点通过几何构造命令得到
- 各点标注方向应错开，避免标签重叠（如三角形三个顶点分别用 lt、lb、rb）
- 使用 GCLC 提供的"构造"而非"描图"——例如线段中点用 `midpoint` 而非直接坐标
- GCLC 代码不要包含 BOM 或特殊字符，使用纯 ASCII/UTF-8 文本
